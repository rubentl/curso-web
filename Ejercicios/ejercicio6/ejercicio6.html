Ejercicio con una imagen de fondo de página

Lunes, 25 de Septiembre de 2006

  1. Cómo poner una imagen de fondo.

    2. "imágenes" o "images". Dentro de ella guardaremos todas las imágenes que vayan a formar parte de las diferentes páginas de nuestro sitio.

  2. Insertar la imagen

    3. Una vez que tenemos la imagen guardada en la carpeta correspondiente a las imágenes, en la etiqueta body añadimos la propiedad background="imagenes/nombreimagen.ext". En esta propiedad le indicaremos el camino de dónde se encuentra la imagen. Le decimos la carpeta y el nombre completo de la imagen (eso quiere decir con la extensión incluída).

  3. Ver el resultado

    4. Para ver el resultado, guardaremos la página web en la carpeta correspondiente y le pondremos extensión .html, para decirle al bock de notas, que lo que guardamos es una web. Accedemos a la carpeta y hacemos doble click encima del archivo que tiene el nombre que acabamos de poner. Se abrirá el Internet Explorer mostrando la página con la imagen de fondo.
